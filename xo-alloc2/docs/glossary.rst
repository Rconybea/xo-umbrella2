.. _glossary:

Glossary
--------

.. glossary::
    FOMO
     | facet object model

    page
     | a (4k) page of virtual memory.
     | O/S manages virtual memory in chunks of this size.

    hugepage
     | large (2MB) VM page; use to reduce page fault expense and TLB pressure.

    THP
     | transparent huge pages

    TLB
     | translation lookaside buffer

    VM
     | virtual memory
